**why? nuxtJ.S**
>기존 Vue.js의 SPA()에서의 단점인 SEO(Search Engine Optimization)을 띄어나게 해줄 SSR(Server Side Rendering)을 권장하고 있으며 리액트의 next.js를 기반으로한 nuxt.js를 사용하고자 합니다.
